
1. > File > New Project > Installed > Visual C# > .NET Core > Class Library (.NET Core)

2. File > Add New Project > Installed > Visual C# > MS Test Test Project (.NET Core)

3. View > Other Window > Package Manager Console

4. In Package Manager Console (CorePaginationExample)
	> Install-Package Microsoft.EntityFrameworkCore
	> Install-Package Microsoft.EntityFrameworkCore.SqlServer
	> Install-Package Microsoft.EntityFrameworkCore.Tools

5. In Package Manager Console (CorePaginationExample.Tests)

	> Install-Package Microsoft.EntityFrameworkCore.InMemory
	
6. Add reference to the class library in models.

7. Create the Database Migrations
	> Add-Migration Initial

8. Done
